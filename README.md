# Linear-Regression
# 🏠 Housing Price Prediction

This project uses Linear Regression to predict house prices based on different features.

## 📌 Features
- Simple Linear Regression (Area vs Price)
- Multiple Linear Regression (All features)
- Data preprocessing (encoding + dummies)
- Model evaluation (MAE, MSE, R²)

## ⚙️ Tech Stack
- Python
- Pandas
- Matplotlib
- Scikit-learn

## 🚀 How to Run

Install dependencies:
    pip install pandas matplotlib scikit-learn

Run the script:
    python your_script_name.py

## 📊 Output
- Performance metrics (MAE, MSE, R²)
- Regression graph
- Sample prediction

## 📂 Dataset
- Housing.csv
